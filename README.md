# SubdomainX
Domains Collector 

# Features 
*  Crt.sh..
*  Baidu..
*  Yahoo..
*  Google..
*  Bing..
*  Ask..
*  Netcraft..
*  DNSdumpster..
*  ThreatCrowd..
*  SSL Certificates..
*  PassiveDNS..
*  Subfinder..
*  Amaas..


# Requirments 
1. sudo apt install subfinder
2. sudo apt install amass
3. sudo apt install assetfinder

# Install

```
go install github.com/zapstiko/subdomainx@latest
```

# Use 
subdomainx -d domain.com
