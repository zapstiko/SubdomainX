# SubdomainX
Domains Collector 

# Features 
*  Crt.sh..
*  Baidu..
*  Yahoo..
*  Google..
*  Bing..
*  Ask..
*  Netcraft..
*  DNSdumpster..
*  ThreatCrowd..
*  SSL Certificates..
*  PassiveDNS..
*  Subfinder..
*  Amaas..


# Requirments 
1. sudo apt install subfinder
2. sudo apt install amaas
3. sudo apt install assetfinder

# Install

```
git clone https://github.com/zapstiko/SubdomainX.git
cd SubdomainX
chmod +x subdomainx
sudo cp subdomainx /usr/local/bin/
```

# Use 
subdomainx domain.com
